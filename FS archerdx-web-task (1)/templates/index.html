Hello! Looks like you're up and running. Here's what to do next.

Modify this Django application such that a BED file can be uploaded here. The uploader should, at minimum:

  • Save the genomic regions specified in the BED file to database via Django's ORM.
  • Render a graphical summary of the regions uploaded using plotly.js. This could be as simple as a pie chart, but pick whatever chart type suits your fancy.
  • Be well tested!

Example BED files can be found here and here. Feel free to modify the existing code however you like, so long as the above requirements are met. This is only a starting point.
